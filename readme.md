#  Resume Skill Gap Analyzer

##  Overview
This project analyzes a candidate's resume against a job description and identifies missing skills using Natural Language Processing (NLP).

##  Features
- Resume vs Job Description matching
- Match score (cosine similarity)
- Skill extraction
- Missing skill detection
- PDF resume upload support

##  Technologies Used
- Python
- Scikit-learn
- NLP (NLTK)
- Streamlit
